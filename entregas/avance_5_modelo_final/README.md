# Avance 5.4 - Modelo final

Esta carpeta contiene la quinta entrega semanal del proyecto academico AURORA.

## Objetivo

Comparar modelos de ensamble con el mejor modelo individual de la fase anterior, seleccionar un modelo final y evaluar su desempeno en datos no vistos mediante metricas y visualizaciones.

## Contenido

- `Avance5.4.ipynb`: libreta principal ejecutable de forma secuencial.
- `data/aurora_ensemble_model_comparison.csv`: comparativa del modelo individual de referencia y modelos de ensamble.
- `data/aurora_final_model_summary.csv`: resumen del modelo final seleccionado.
- `data/aurora_final_model_predictions.csv`: predicciones del modelo final.
- `data/aurora_final_model_feature_importance.csv`: importancia de variables por permutacion.
- `data/aurora_final_model_threshold_analysis.csv`: analisis de umbral.
- `data/figures/`: graficas generadas por la libreta.

## Privacidad

Todos los datos son sinteticos y se usan solo con fines academicos. La entrega no incluye informacion real, repositorios, URLs, tecnologias internas, credenciales, procedimientos operativos ni nombres de sistemas.
